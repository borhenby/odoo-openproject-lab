# Odoo + OpenProject API Lab

AWS-based practice lab for integrating Odoo with OpenProject using APIs.

## Stack
- AWS EC2
- Docker
- Docker Compose
- Odoo
- OpenProject
- PostgreSQL

## Services
- Odoo: port 8069
- OpenProject: port 8080

## Goal
Practice API-based integration between Odoo and OpenProject in a cost-effective AWS lab.
